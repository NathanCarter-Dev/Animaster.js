ANIMASTER

Build fast animations with Animasters vast pure css library

Usage

Installation

To install Animaster, simply click Download and then drop the css file in your project directory. You will then have access to all the Animaster classes by using the <link> tags in your html to find Animaster.css

Basic usage

After installing Animaster.css, to add an animation, choose animation name followed by the speed you want your animation to play at.

<button class="blur speed-slow">Button Example</button>

Using speeds

Speeds classes consist of fast(0.5s), medium(1s), slow(1.5s), veryfast(0.25s), veryslow(2s) use case is speed followed by speed intensity e.g speed-veryfast

Using values

values are added onto certain effects. If shown in effect list the various values below can be added onto effects followed by -

opacity: 0-100/100-0/100-50/0-50. e.g opacity-100-50

pixels: 20/40/60/80/100. e.g flyinright-60
